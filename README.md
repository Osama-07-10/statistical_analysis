# Titanic Data Analysis Project

This project is an exploratory data analysis (EDA) performed on the Titanic dataset using Python and Jupyter Notebook.  
The goal is to understand the data, clean it, and extract meaningful insights through statistics and visualizations.

---
## Exploratory Data Analysis (EDA)
The notebook includes multiple visualizations such as:
- Boxplots for numerical features (Age, Fare, etc.)
- Distribution analysis of passengers
- Survival analysis based on different attributes
- Correlation matrix between key numerical variables

 ## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
